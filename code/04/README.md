- Massiivid

 - Massiivi pikkus, ehk elementide arv massiivis(massiiv.length)
 - Massivi elementide indeksid algavad alati nullist
 - Viimase elemendi indeks on (length -1)

  - Funktsiooni argumendid (sulgudes) - põhimõtteliselt muutujad funktsiooni sees
